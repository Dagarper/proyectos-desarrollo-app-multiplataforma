Integración de JavaHelp en Aplicaciones Java

Para usar JavaHelp en una aplicación Java, sigue estos pasos:

1. Importa las clases necesarias

import javax.help.HelpBroker;
import javax.help.HelpSet;
import javax.help.HelpSetException;

2. Carga el sistema de ayuda

try {
    ClassLoader cl = getClass().getClassLoader();
    URL hsURL = HelpSet.findHelpSet(cl, "help/help_set.hs");
    HelpSet helpSet = new HelpSet(cl, hsURL);
    HelpBroker helpBroker = helpSet.createHelpBroker();
    helpBroker.enableHelpOnButton(button, "introduccion", helpSet);
} catch (HelpSetException e) {
    e.printStackTrace();
}

3. Asocia un botón para abrir la ayuda

Utiliza `helpBroker.enableHelpOnButton(...)` para asociar un botón con la ayuda.
