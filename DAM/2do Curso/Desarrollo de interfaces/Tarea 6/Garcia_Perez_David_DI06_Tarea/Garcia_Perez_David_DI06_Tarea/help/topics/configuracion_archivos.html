Configuración de Archivos de Mapa

JavaHelp utiliza varios archivos XML para estructurar la ayuda:

1. Archivo `map_file.jhm`

Define la correspondencia entre identificadores y archivos HTML:

<map version="2.0">
    <mapID target="introduccion" url="topics/introduccion.html"/>
    <mapID target="instalacion" url="topics/instalacion.html"/>
</map>

2. Archivo `toc.xml`

Define la tabla de contenidos:

<toc version="2.0">
    <tocitem text="Introducción" target="introduccion"/>
    <tocitem text="Instalación" target="instalacion"/>
</toc>

3. Archivo `index.xml`

Define los términos para el índice:

<index version="2.0">
    <indexitem text="Introducción" target="introduccion"/>
</index>
